# Unity 3D Cube Runner

## Overview

A simple 3D Unity game developed to explore Unity game development concepts, including player movement, physics, and scene design. The project features a controllable cube navigating a 3D environment with obstacle interaction.

## Features

* 3D player movement
* Physics-based gameplay
* Basic obstacle navigation
* Unity scene management
* C# scripting

## Tech Stack

* Unity
* C#
* Visual Studio

## Project Structure

* `Assets/` – Game assets, scripts, scenes, and materials
* `Packages/` – Unity package dependencies
* `ProjectSettings/` – Project configuration

## Status

Completed as a learning project for Unity game development.
